DATASET PURIFIER

About

In any Machine Learning process, Data Preprocessing is the primary step wherein the raw/unclean data are transformed into cleaned data, So that in the later stage, machine learning algorithms can be applied. All you have to do is just input a raw data(CSV file), this website will clean your data and return you the cleaned dataframe on which further you can apply feature engineering, feature selection and modeling.

  • What this does?
    • Cleans special character
    • Removes duplicates
    • Fixes abnormality in column names
    • Imputes the data (categorical & numerical)
